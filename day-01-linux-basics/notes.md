# Day 1 – System Setup & Basic Commands

## What I Learned
- What a Linux filesystem is
- How to navigate the terminal
- Basic commands for file and directory management

## Key Commands
- `pwd` – show current directory
- `ls` – list files
- `cd` – change directory
- `mkdir` – create directory
- `touch` – create files
- `rm` – remove files
