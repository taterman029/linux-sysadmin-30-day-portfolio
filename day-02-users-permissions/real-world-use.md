Real-World Use Cases – Day 2

## Why Users & Permissions Matter

Linux permissions are critical in system security and production environments.

---

## Server Security

- Prevent unauthorized access to system files
- Protect sensitive directories like /etc and /root
- Secure SSH keys in ~/.ssh

---

## Web Servers

- Control access to /var/www
- Ensure only web services can modify files
- Prevent unauthorized file uploads or edits

---

## Multi-User Systems

- Separate users by role (admin, developer, service accounts)
- Limit damage from compromised accounts
- Enforce least privilege principle

---

## Production Risk Example

Incorrect permissions can lead to:
- Data leaks
- Unauthorized system access
- Service compromise
